# Marketing Assets

Lead magnet positioning, landing page copy, email swipe copy, and video content plan.

## Purpose

This folder supports distribution of the 90-Day Ecommerce Cash Flow Planner and Ecommerce Cash Flow Forecaster GPT.

## Files

- `lead-magnet-description.md` — Offer description, positioning, benefits, CTAs, social copy, and compliance-safe disclaimer.
- `landing-page-copy.md` — Conversion-ready landing page copy for the planner.
- `email-swipe-copy.md` — Lead magnet delivery and nurture email sequence.
- `video-content-plan.md` — Long-form, Shorts/Reels, funnel, thumbnail, and repurposing plan.

## Recommended Usage

- Use these assets for landing pages, email funnels, social posts, videos, and partner promotions.
- Do not upload these to the CustomGPT knowledge base unless the GPT will help with marketing content.
- Keep compliance language intact around funding and financial planning.

## Repository Context

This folder is part of the `JFeimster/ecommerce-cash-flow` project for the Ecommerce Cash Flow Forecaster GPT and the 90-Day Ecommerce Cash Flow Planner.

## Upload Rule

Use GitHub as the canonical version-controlled source, Google Drive as the editable/shareable workspace, and CustomGPT uploads as approved release builds only.
