# Prompt Library

Reusable prompts for weekly cash flow analysis, scenario planning, and funding memo generation.

## Purpose

This folder gives users and the GPT repeatable prompt workflows for turning spreadsheet data into useful decisions.

## Files

- `weekly-analysis-prompts.md` — Prompts for weekly cash reviews, risk dashboards, ad spend checks, inventory pressure, and cash preservation.
- `scenario-planning-prompts.md` — Prompts for Conservative/Base/Optimistic scenarios and stress tests.
- `funding-memo-prompts.md` — Prompts for funding memos, working capital summaries, readiness checklists, and advisor briefs.

## Recommended Usage

- Use these prompts inside the GPT, spreadsheet instructions, or marketing follow-up.
- Keep prompt files separate from core knowledge files so the GPT brain stays clean.
- Upload to CustomGPT knowledge only if you want the GPT to expose prompt libraries to users.

## Repository Context

This folder is part of the `JFeimster/ecommerce-cash-flow` project for the Ecommerce Cash Flow Forecaster GPT and the 90-Day Ecommerce Cash Flow Planner.
