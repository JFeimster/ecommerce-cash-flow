# Release Package v1.0

Manifest and checklist for the first release of the Ecommerce Cash Flow Forecaster GPT knowledge system.

## Purpose

This folder documents what belongs in the v1.0 release and how to upload/test it.

## Files

- `customgpt-upload-manifest.md` — Upload strategy, required files, optional files, GPT instructions summary, and test cases.
- `upload-checklist.md` — Pre-upload, Drive, GitHub, CustomGPT, compliance, and post-upload testing checklist.

## Recommended Usage

- Use this folder as the release control point for v1.0.
- Do not upload release files to the GPT knowledge base unless you want the GPT to explain its own deployment package.
- Use the checklist before updating CustomGPT knowledge files.

## Repository Context

This folder is part of the `JFeimster/ecommerce-cash-flow` project for the Ecommerce Cash Flow Forecaster GPT and the 90-Day Ecommerce Cash Flow Planner.

## Upload Rule

Use GitHub as the canonical version-controlled source, Google Drive as the editable/shareable workspace, and CustomGPT uploads as approved release builds only.
